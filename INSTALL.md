# Quantheo Desktop Installation

Welcome to Quantheo Desktop! This guide provides instructions on how to install the desktop application.

## Prerequisites
- Windows 10 or Windows 11 (64-bit)

## Installation Steps
1. Locate the downloaded installer executable: `Quantheo Setup 1.0.0.exe`.
2. Double-click the executable to start the installation.
3. **Accept the License Agreement**: You will be presented with the End User License Agreement (EULA). You must accept the Terms and Conditions to proceed.
4. Follow the on-screen prompts in the setup wizard to choose an installation directory (or use the default).
5. Once installation finishes, you can choose to run the application immediately or launch it later from the Start Menu shortcut (`Quantheo`).

## First Launch & Trial
When you first launch Quantheo, you will be granted a 30-day trial of all premium features.
- After 30 days, the application will lock advanced features, allowing only basic PDF viewing and Word <-> PDF conversions.
- To unlock the full suite of features after the trial expires, you must apply a valid License Key in the **License & Settings** tab. Licenses are valid for one year.

## Troubleshooting
- **Backend fails to start**: Make sure port 8080 is not already in use by another application. You can change the port in the **Settings** tab.
- **Can't find the app**: Look for the Quantheo icon in your system tray (bottom right corner of the taskbar). The application minimizes to the tray to stay running in the background.
