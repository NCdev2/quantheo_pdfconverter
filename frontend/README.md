# StirlingPDF Frontend

This is the mobile and web frontend for StirlingPDF.

## Install

```bash
cd frontend
npm install
```

## Run locally

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Capacitor commands

```bash
npm run cap:init
npm run cap:sync
```

Once built, use Capacitor to add Android support:

```bash
npx cap add android
```

Then open the native Android project:

```bash
npx cap open android
```
