# StirlingPDF Backend

This backend provides a simple Express API for the StirlingPDF mobile/web frontend.

## Install

```bash
cd backend
npm install
```

## Run in development

```bash
npm run dev
```

## Build for production

```bash
npm run build
npm start
```

## Endpoints

- `GET /health` — health status
- `GET /info` — API metadata
- `POST /convert` — placeholder for PDF conversion
